# AI Image Captioning

This project is a full-stack AI application that generates captions for images using a pretrained deep learning model.

## 🚀 Features
- Upload image and generate captions
- Uses pretrained model (BLIP)
- Optional BLEU score evaluation
- Clean UI with HTML, CSS, JS
- Flask backend API

## 🧠 Tech Stack
- Python (Flask)
- Hugging Face Transformers
- HTML, CSS, JavaScript

## ▶️ Run Locally

```bash
pip install -r requirements.txt
cd backend
python app.py
